3-1

☆ CSSの基礎知識

CSSはHTMLの表示を制御するための言語

CSS（Cascading Style Sheets）
HTMLにスタイル機能を提供し、表示を制御して見た目を整形するための言語
フォントを変える、複雑なレイアウトを組むetc

CSSの基本的な仕組み

関連するHTMLドキュメントの中から要素を選択し、その選択した要素に適したスタイルを適用して表示を変更する
例）
見出しのテキスト{ 箇条書き; 箇条書き; }

CSSの基本的な書式と各部の名称

例）
h1{ background-color:#FFFAFO; color:#FFF; }

ルール（ルールセット）
セレクターと宣言ブロックがセットになったもの（つまり全体）
CSSのルールはアルファベット、：、；、スペースなどの記号も含めて全て半角で記述する

セレクター
HTMLドキュメントから特定の要素を選択する部分
「パターン」と呼ばれる、要素を選択する条件が定義されている
例）h1

宣言ブロック
開始波カッコ（{）から終了波カッコ（}）までの部分
この中にセレクターで選択した要素に適用するスタイルを記述する
例）{ background-color:#FFFAFO; color:#FFF; }

プロパティ
CSSで操作できるスタイルそれぞれに「プロパティ」が定義されている（「フォントを指定する」「テキスト色を変更する」）
例）color

プロパティ値
プロパティに設定する値
色のプロパティであれば、値に色を設定する
プロパティとプロパティ値の間にはコロン（:）が必ず入る
プロパティ値の後ろには必ずセミコロン(;)が付く
例）#FFF

宣言（スタイル宣言）
プロパティとその値をまとめた呼称
必ずプロパティとプロパティ値はセットで記述する
例）background-color:#FFFAFO;

コメント文

/**これがCSSのコメント/

@ルール

CSSにはセレクターのない「@ルール」というものがある

@charsetルール
CSSの文字コードを指定する
原則UTF‐8
例）@charset "UTF-8";

@importルール
CSSファイルから別のCSSファイルを読み込むことができる
例）@import 読み込むCSSファイルへのパス

@mediaルール
特定の条件を満たすときだけ適用されるCSSルールを作成することができる
例）@media(max-width: 500px){ ここにCSSルールを記述 }
