JS note

基本構文

変数宣言

let message = "こんにちは、JavaScript!";
console.log(message);

  1. 変数
    • let
      再代入可能な変数を宣言する 現在最も一般的に使用されている
    • const
      値を入力すると変更できない確定数を宣言する
    • var
      古い書き込みで再挿入可能だが、スコープの扱いが特殊なので現在はあまり推奨されない
  2. コードの動作
    • let message = "こんにちは、JavaScript!";
      変数messageに文字列"こんにちは、JavaScript!"を入力
    • console.log(message);
      コンソールにmessageの内容を表示
  3. 結果
    • デベロッパーツールを開き、Consoleタブを見ると「こんにちは、JavaScript!」と表示される

条件分岐（if文）

let age = 20;
if (age >= 18) {
console.log("成人です");
} else {
console.log("未成年です");
}

  1. 条件分岐の基本構文

    2. if (条件) {
    // 条件がtrueの場合に実行される処理
    } else {
    // 条件がfalseの場合に実行される処理
    }

  2. コードの動作
    • let age = 20;
      年齢を表す変数ageを20に設定
    • if (age >= 18)
      条件age >= 18（年齢が18歳以上）という条件を設定
    • 条件がtrueの場合、console.log("成人です");が実行される
    • 条件がfalseの場合、else内がconsole.log("未成年です");実行される
  3. 結果
    • ageが20なので「成人です」と表示される

ループ

for (let i = 1; i <= 5; i++) {
console.log("カウント: " + i);
}

  1. forループの基本構文

    2. for (初期化; 条件; 更新) {
    // 繰り返す処理
    }

    • 初期化
      ループ用の変数を宣言（例: let i = 1）
    • 条件
      ループを続ける条件を書く（例: i <= 5）
    • 更新
      ループのたびに変数を更新（例: i++）
  2. コードの動作
    • for (let i = 1; i <= 5; i++)
      1から5まで、1ずつ増やしながらループを実行
    • ループ内でconsole.log("カウント: " + i);を実行
  3. 結果
    • コンソールに次のように表示される

      カウント: 1
      カウント: 2
      カウント: 3
      カウント: 4
      カウント: 5

関数

function greet(name) {
return "こんにちは、" + name + "!";
}
console.log(greet("田中"));

  1. 関数の基本構文

    2. function 関数名(引数) {
    // 実行する処理
    return 戻り値;
    }

    • function greet(name)
      greetという関数を定義し、というname引数を受け取る
    • return "こんにちは、" + name + "!";
      引数nameを使い、「こんにちは、[名前]!」という文字列を戻り値として返す
    • console.log(greet("田中"))
      関数greetを呼び出し、引数に"田中"渡します。その結果をコンソールに表示
